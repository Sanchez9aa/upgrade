Prueba Técnica: Reforzamiento de JavaScript, HTML5 y CSS
Objetivo: Crear una aplicación web que muestre una lista de usuarios y sus detalles.

Requerimientos
HTML5
Estructura básica: Usa la estructura de HTML5, incluyendo <!DOCTYPE>, <html>, <head>, y <body>.
Semántica: Usa etiquetas semánticas como <header>, <footer>, <nav>, <section>, <article>, etc.
Formulario: Incluye un formulario que permita agregar nuevos usuarios con los siguientes campos: Nombre, Correo electrónico y Número de teléfono.
CSS
Reset/Normalize: Incluye un reset o normalize.css para asegurarte de que los estilos se vean consistentemente en diferentes navegadores.
Flexbox: Utiliza Flexbox para:
Centrar vertical y horizontalmente el título de tu aplicación.
Mostrar la lista de usuarios en una cuadrícula responsiva (por ejemplo, 1 columna en móviles, 2 en tablets, 3 en desktop).
Estilos básicos: Asegúrate de que tu aplicación sea atractiva y legible. Juega con colores, fuentes y espaciado.
Responsividad: La aplicación debe verse y funcionar bien en dispositivos móviles y de escritorio.
JavaScript
Manipulación del DOM: Cuando se agregue un nuevo usuario a través del formulario, añade este usuario a la lista sin necesidad de recargar la página.
Validación: Antes de agregar un usuario, verifica que todos los campos estén llenos y que el correo electrónico tenga un formato válido.
Interacción: Permite a los usuarios hacer clic en un usuario para ver más detalles (puedes mostrar esto en un modal o en un panel lateral).
Almacenamiento local: Usa localStorage para guardar y recuperar la lista de usuarios incluso después de recargar la página.
BONUS
Animaciones: Agrega animaciones sutiles cuando se añaden usuarios o se muestran detalles.
Compatibilidad: Asegúrate de que tu aplicación sea compatible con los últimos 2-3 versiones de los principales navegadores (Chrome, Firefox, Safari).
SASS o LESS: Si estás familiarizado, utiliza preprocesadores CSS como SASS o LESS para organizar y optimizar tus estilos.
Criterios de Evaluación
Funcionalidad: La aplicación debe funcionar como se describe.
Código limpio: Tu código debe ser legible, bien organizado y seguir buenas prácticas.
Diseño: La aplicación debe ser visualmente atractiva y ofrecer una buena experiencia de usuario.
¡Buena suerte! Esperamos ver cómo aplicas tus conocimientos en JavaScript, HTML5 y CSS para crear una aplicación robusta y atractiva.
