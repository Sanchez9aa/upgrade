Prueba Técnica: Construcción de una Página Web Interactiva
Objetivo: Desarrollar una página web que permita al usuario agregar, eliminar y marcar tareas.

Paso 1: Preparación del entorno
Crea una nueva carpeta en tu computadora llamada tareas-web.
Dentro de esa carpeta, crea tres archivos: index.html, estilos.css y scripts.js.

Paso 2: Estructura HTML
Abre index.html y escribe la estructura básica de un documento HTML5.
Dentro del <body>, agrega un título <h1> que diga "Mi Lista de Tareas".
Debajo del título, crea un <input> donde el usuario podrá escribir la tarea y un botón <button> que diga "Agregar".
A continuación, crea una lista <ul> donde se mostrarán las tareas.

Paso 3: Estilos CSS
Abre estilos.css y agrega estilos para centrar el título y los elementos en la página.
Estiliza el botón para que se vea más atractivo (puedes cambiar el color de fondo, la fuente, etc.).
Añade estilos para las tareas. Por ejemplo, cuando una tarea esté marcada como completada, podría tener un tachado.

Paso 4: Funcionalidad JS
Abre scripts.js.
Escribe una función que añada una tarea a la lista cuando se haga clic en el botón "Agregar". La tarea debe ser el valor del input.
Añade la posibilidad de marcar una tarea como completada al hacer clic en ella. Puedes hacerlo añadiendo una clase CSS que tenga un estilo de texto tachado.
Añade la posibilidad de eliminar una tarea al hacer doble clic en ella.

Paso 5: Integración
Vincula estilos.css y scripts.js a tu index.html.
Abre index.html en tu navegador y verifica que todo funcione correctamente: agregar tareas, marcarlas como completadas y eliminarlas.

BONUS:
Si sientes que fue muy sencillo y quieres un desafío adicional:

Añade la funcionalidad de guardar las tareas en el localStorage del navegador para que, al recargar la página, no se pierdan.
Implementa un contador que muestre cuántas tareas hay pendientes y cuántas están completadas.
Con esta prueba, podrás practicar y reforzar tus habilidades en HTML, CSS y JS. Una vez que la completes, te recomiendo que busques retroalimentación o incluso intentes agregar más funcionalidades por tu cuenta. ¡Buena suerte!
